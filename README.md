# PHP CRUD Application with Authentication

A beginner-friendly PHP project demonstrating core web development concepts such as
CRUD operations, user authentication, session handling, and form validation using PHP and MySQL.

#Features
- User Registration & Login
- Secure Authentication using PHP Sessions
- CRUD Operations (Create, Read, Update, Delete)
- Server-side Validation
- Admin & User Dashboard
- Basic JavaScript for UI interactions

#Tech Stack
- PHP (Core PHP, Procedural)
- MySQL
- HTML5, CSS3
- JavaScript
- Apache (XAMPP)
